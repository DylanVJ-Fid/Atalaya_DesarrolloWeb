# Atalaya - Plataforma de Comercio Electrónico

## Descripción del proyecto

Atalaya es una plataforma web de comercio electrónico orientada a pequeñas y medianas empresas que desean gestionar la venta de productos en línea, administrar usuarios, controlar pedidos y mejorar su presencia digital.

El sistema permitirá a los clientes registrarse, iniciar sesión, consultar productos, agregar artículos al carrito y confirmar compras. Además, los administradores podrán gestionar productos, categorías, usuarios, roles y pedidos.

## Tecnologías utilizadas

- Java
- Spring Boot
- Thymeleaf
- Bootstrap
- HTML5
- CSS3
- Hibernate / JPA
- Base de datos relacional
- GitHub

## Funcionalidades principales

- Registro de usuarios
- Inicio de sesión
- Recuperación de contraseña
- Catálogo de productos
- Búsqueda y filtrado de productos
- Carrito de compras
- Confirmación de pedidos
- Historial de pedidos
- Gestión de productos
- Gestión de categorías
- Gestión de usuarios y roles
- Reportes básicos de ventas
